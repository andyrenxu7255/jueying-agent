# 重构速查表

| 操作 | 命令/路径 | 说明 |
|------|-----------|------|
| 切换到 fork 分支 | `git checkout fork-audit-optimized` | |
| 回看 Phase 1 差异 | `git diff master..fork-audit-optimized -- libs/shared/` | |
| 回看全部差异 | `git diff master..fork-audit-optimized --stat` | |
| 编译检查 | `npx tsc --noEmit` | |
| 切换回 master | `git checkout master` | |
| 模块文件列表 | `libs/shared/src/{utils,llm,db,channel,server,intent}/` | |
